# USB Monitoring Tool Kit

A comprehensive forensic and monitoring toolkit for USB devices. This project provides an administrative console built with Python and Tkinter to track connections, manage clients, and log system events in real-time.

---

## üöÄ Features

- **Live Monitoring**: Real-time tracking of connected USB devices and client activity.
- **Admin Console**: User-friendly GUI for server control and client management.
- **Secure Logging**: Persistent logs of all server and client communications.
- **Client Control**: Ability to authenticate and manage (kick) connected clients.

## üõ†Ô∏è Installation & Setup

Follow these steps to get the toolkit running on your local machine:

1. **Prerequisites**:
   - Ensure you have **Python 3.x** installed.
   - Tkinter is usually included with Python, but if not:
     - Windows: Included by default.
     - Linux/Mac: Install via `sudo apt-get install python3-tk`.

2. **Clone the Project**:
   ```bash
   git clone <your-repository-url>
   cd usb-monitoring-tool-kit
   ```

3. **Run the Admin Program**:
   Execute the main server script:
   ```bash
   python admin_program.py
   ```
